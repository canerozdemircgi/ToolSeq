ToolSeq is a pack of useful, high quality plugins and scripts which has 15 tools that can simplify and reinforce 3D modeling pipeline for Autodesk Maya

They are written with Python and Qt for the best maintainability, flexibility and version compatibility

Bottleneck parts are implemented with OpenMaya SDK, other parts are implemented with maya_cmds

All functionality of tools are compatible with internal Maya Undo and Redo stack

Users are supported with 23 tutorial videos including subtitles that have total 2 hours 31 minutes length

ToolSeq is published in Autodesk Store and Highend3D
